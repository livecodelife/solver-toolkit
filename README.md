## 🚀 Quick Start

1. **Clone the repo**  
   ```bash
   git clone https://github.com/your-username/solver-toolkit.git
   cd solver-toolkit
   ```
2. **Explore the AI configs**

   ```bash
   cd ai/
   ls
   ```

3. **Adopt & Customize**

   * Copy any configuration or script into your own environment.
   * Adjust parameters, API keys, or paths to fit your setup.
